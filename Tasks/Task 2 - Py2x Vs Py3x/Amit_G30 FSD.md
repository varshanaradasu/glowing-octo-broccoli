
Differences Between Python 2.x and Python 3.x

1) Print Statement
   Python 2: print "Hello"
   Python 3: print("Hello")

2) Integer Division
   Python 2: 5 / 2 = 2
   Python 3: 5 / 2 = 2.5

3) Unicode Support
   Python 2: Default string type = ASCII
   Python 3: Default string type = Unicode
   ASCII = only English characters
   Unicode = characters of all languages

4) Input Function
   Python 2: raw_input() for string input
   Python 3: input() for all input

5) xrange vs range
   Python 2: xrange() is used for loops (memory efficient)
   Python 3: range() works like xrange (memory efficient by default)

6) Error Handling Syntax
   Python 2: except Exception, e
   Python 3: except Exception as e

7) Library Support
   Python 2: Official support ended in 2020
   Python 3: Actively supported with new libraries and updates

Main Differences Summary:
- print changed from a statement (Python 2) to a function (Python 3).
- Division changed: Python 3 always returns a float for '/'.
- Unicode became default in Python 3, making string handling easier.
