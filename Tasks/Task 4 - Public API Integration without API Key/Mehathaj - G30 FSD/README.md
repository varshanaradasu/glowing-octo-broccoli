# Weather App

A simple weather application where users can enter a city name and get real-time weather information using the Open-Meteo API (no API key required).

## Features
- Search weather by entering a city name  
- Shows temperature, humidity, wind speed, pressure, visibility, cloud cover, UV index  
- Air Quality Index (PM2.5, PM10, CO, NO₂, SO₂, O₃)  
- Responsive UI  
- Clean and modern design  

---

## How to Use
1. Enter a city name in the search bar  
2. Click **Search**  
3. The weather details will load instantly  

---

## Notes
- No API key required  
- Works for all countries  
- Built with HTML, CSS, and JavaScript
