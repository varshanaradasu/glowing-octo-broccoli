#Difference between python v2 and python v3:

Release & Support:
Python 2
Released in 2000
Reached End of Life (EOL) on January 1, 2020
No updates, no security fixes
Mostly used in old projects

Python 3
Released in 2008
Active support & updates
Recommended for all new projects
Modern libraries only support Python 3

    1. print statement:
python 2:       print “hello”
python 3:       print(“hello”)

    2. integer division:
python 2:       5/2     #output:2 (floor division)
python 3:       5/2     #output:2.5

    3. Unicode support:
python 2:       str=bytes ----→u”hello”
python 3:       str= unicode by default

    4. range:
python 2:       xrange(5),range(5)
python 3:       range(5)

    5. error handling :
python 2:       except Exception ,e:
                     print e
python 3:       except Exception as e:
                     print(e)
